### TL;DR![Uploading waitless.png…]()


　近辺の飲食店の混雑度を可視化するマップアプリ

### こだわりリスト

- 地名を検索すると周辺の飲食店を一括表示
- 予測される混雑度に応じて表示ピンの色が変化
- LightGBLを用いた自作の混雑度予測モデル

### 使用技術

- CSS
